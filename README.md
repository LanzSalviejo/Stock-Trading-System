A stock trading management system built with React (Vite) and Flask.

### Frontend Setup

# Install dependencies
npm install

# Start development server
npm run dev


### Backend Setup

# Create and activate virtual environment
python -m venv venv

# Install dependencies
pip install -r requirements.txt

# Start server
python app.py
